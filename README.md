# Yin Editor（YE）终端文本编辑器

## 设计
文档：被YE加载的文件
终端：由两部分存在
  1. 终端区域：和整个文档在宽高上一模一样的终端，即终端与文档是完完全全重叠在一起的，两者均没有丝毫多余。
  2. 可视区域：用户肉眼所见的终端。

**图元 Cell**：一个视觉符号就是一个图元。
**图行 Line**：一个或多个图元组成的行。

当文件被加载成文档时，文件中的每一个字符就会被加载成一个图元，每一行字符则加载成一个图行。
**文档坐标**：(line_idx, cell_idx) (图行索引, 图元索引)

终端是不管图元这种的，它是以字节为基本单位的，终端一格的宽度=1字节、高度=1字节。
**终端坐标**：(row, col) (RowIdx, ColIdx)

**光标 creat**：基于文档坐标，移动光标时，是按照图元、图行的索引来计算光标位置的。
**滚动偏移量 scroll_offset**：基于终端坐标，以终端区域的左上角(0, 0)为锚点来计算偏移量的。

滚动文档：光标的文档坐标会转为终端坐标，和滚动偏移量进行一些计算，进而算出可视区域应当处在**终端区域**的哪个位置。当文档发生滚动时，滚动偏移量也会发生改变。
